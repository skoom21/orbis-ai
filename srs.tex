\section{Overall System Description}

\subsection{Project Background}

Modern travel planning suffers from significant fragmentation. Users must navigate multiple platforms for flights, accommodations, and activities, manually reconcile pricing and policies, and risk missing optimal options due to information overload.

\begin{infobox}[Problem Statement]
\begin{itemize}[leftmargin=*, itemsep=0.3em]
    \item \textbf{Fragmented Services:} Inefficient planning across multiple disconnected platforms
    \item \textbf{Lack of Personalization:} Limited real-time adaptation to user preferences
    \item \textbf{Complexity:} Difficult comparison and booking across providers
    \item \textbf{Trust Issues:} Need for transparent, ethical AI in handling sensitive user data
\end{itemize}
\end{infobox}

\textbf{Orbis AI Solution:} An autonomous travel agent leveraging agentic AI to understand natural language queries, decompose them into structured tasks, and integrate results from real-time APIs to deliver personalized, optimized travel itineraries.
\newpage
\subsection{Project Scope}

Orbis AI aims to revolutionize travel planning through intelligent AI orchestration, providing transparency, efficiency, and deep personalization.

\subsubsection{Included Functionalities}

\begin{itemize}[leftmargin=*, itemsep=0.3em]
    \item \textbf{Conversational Interface:} Natural language-based travel planning via chat
    \item \textbf{AI Orchestration:} Intelligent coordination of flights, hotels, and itineraries
    \item \textbf{API Integration:} Amadeus, Booking.com, Stripe sandbox, Google Maps
    \item \textbf{Personalization Engine:} Preference learning through questionnaires and embeddings
    \item \textbf{Real-Time Support:} Dynamic trip management, notifications, and change handling
    \item \textbf{Sandbox Payments:} Demo-only transactions with no real charges
    \item \textbf{MARL Optimization:} Multi-Agent Reinforcement Learning for optimal planning
    \item \textbf{Multi-Modal Summaries:} LLM-powered trip visualization with text, maps, and images
    \item \textbf{Dynamic Re-Planning:} Real-time constraint solver for itinerary adjustments
    \item \textbf{Future Features:} Admin panel for analytics, logs, and system audits
\end{itemize}

\subsubsection{Excluded Functionalities}

\begin{itemize}[leftmargin=*, itemsep=0.3em]
    \item Car rentals and travel insurance services
    \item Corporate travel management and compliance rules
    \item Production-grade ticketing with full regulatory compliance
\end{itemize}

\subsection{Not In Scope}

\begin{itemize}[leftmargin=*, itemsep=0.3em]
    \item Real-time API response verification beyond sandbox environments
    \item Production-scale user load handling
    \item Manual booking verification or full production compliance
\end{itemize}

\subsection{Project Objectives}

\subsubsection{User Functions}

\begin{enumerate}[leftmargin=*, itemsep=0.3em]
    \item Initiate trip planning through natural language conversations
    \item Receive AI-curated travel bundles (flights, hotels, activities)
    \item Interactively refine itineraries based on personal preferences
    \item Execute test bookings with sandbox payment processing
    \item Export itineraries to calendar format (ICS)
    \item Receive automated email notifications
    \item Access real-time support for trip modifications
\end{enumerate}

\subsubsection{Success Criteria}

\begin{highlightbox}
\textbf{Core Success Metrics:}
\begin{itemize}[leftmargin=*, itemsep=0.3em]
    \item User successfully signs in and initiates trip request
    \item System generates curated flight and hotel bundles
    \item Realistic day-by-day itinerary created within budget constraints
    \item Test payment processed with booking confirmation
    \item ICS calendar export successfully generated
    \item Change requests processed with updated options
\end{itemize}
\end{highlightbox}

\textbf{Advanced Features:}

\begin{itemize}[leftmargin=*, itemsep=0.3em]
    \item \textbf{MARL Implementation:} Reinforcement learning agents dynamically negotiate between flight, hotel, and itinerary agents to optimize cost, travel time, and user satisfaction
    \item \textbf{Multi-Modal Summaries:} LLM-powered generation combining text, annotated maps, and landmark images for rich trip visualization
    \item \textbf{Real-Time Optimization:} Constraint solver maintains feasible itineraries during dynamic changes (delays, weather, preference updates)
\end{itemize}

\subsubsection{Admin Functions (Future)}

\begin{itemize}[leftmargin=*, itemsep=0.3em]
    \item Access observability dashboards for agent execution monitoring
    \item Audit user interactions and system performance metrics
\end{itemize}
